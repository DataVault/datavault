File Transfer Manager SDK for Java for Oracle Cloud Storage

The File Transfer Manager SDK for Java provides a Java API for managing the file transfers to Oracle Cloud Storage. This SDK enables the developers to upload to or download from files Oracle Storage Cloud.

It manages the connections with cloud storage, threads and RESTful communication with the service, making it very simple to use. It supports Static Large Objects as well, by automatically segmenting the files before uploading them. 

Features:

The SDK includes the following:

	- File Transfer Manager API Java library
	- Sample Code
	- Third-party libraries
	- Javadoc
